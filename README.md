## üöÄ Running Locally

1. Clone the repository
``` bash
git clone ...
```

2. Install dependencies using pnpm
``` bash
pnpm install
```

3. Copy the `.env.example` to `.env` and update the variables.
``` bash
cp .env.example .env
```

4. Start the development server
``` bash
pnpm run dev
```
